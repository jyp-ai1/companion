# MVP 검증 실험 계획서

> Project AGEZERO / 이음 — 파일럿 검증 + 정부지원사업 증거 데이터

---

## 1. 검증 목적

**MVP는 앱 출시가 아니라 가설 검증이다.**

| 가설 | 검증 방법 |
|------|-----------|
| H1: 시니어는 활동·관계를 원한다 | 가입 동기 인터뷰, 모임 참여율 |
| H2: 성향 파악 → 참여율 ↑ | 이음 타입 완료 vs 미완료 참여율 비교 |
| H3: 추천 → 탐색 부담 ↓ | 추천 클릭률, 검색 없이 참여율 |

## 3. 모집 방법

### 3-1. 1차 (지인·커뮤니티) — 30명

- 본인 네트워크, 지인 부모님
- 카카오 오픈채팅 / 단톡
- **비용:** 0원

- 하남시 경로당 / 복지관 / 시니어센터
- 포스터 + QR (companion-sandy.vercel.app)
- **협력 요청:** "활동 친구 찾기 무료 서비스 테스트"

### 3-3. 3차 (소규모 광고) — 20명

- 네이버 밴드 / 지역 커뮤니티
- Facebook 50+ 타겟 (소액)

## 4. 실험 프로토콜

### Week 1: 온보딩 검증

| 측정 | KPI | 방법 |
|------|-----|------|
| 가입 완료 | 100명 | Supabase auth.users count |
| 프로필 완료 | 90%+ | onboarding_completed |
| 이음 타입 완료 | **80%+** | test_completed_at |
| 완료 시간 | 3분 이내 | 테스트 시작~완료 timestamp |

### Week 2~3: 추천·참여 검증

| 측정 | KPI | 방법 |
|------|-----|------|
| 추천 클릭 | 50%+ | 결과→모임상세 event |
| 참여 신청 | **40%+** | participations / 추천 노출 |
| 실제 참석 | 60%+ | admin confirmed |

### Week 4~6: 관계·재참여 검증

| 측정 | KPI | 방법 |
|------|-----|------|
| 후기 제출 | 70%+ | reviews / completed meetups |
| 만족도 | **4.5+** | avg(rating) |
| 재참여 | **30%+** | 2회+ participation |
| 새 사람 만남 | 50%+ | review.met_new_people = true |

## 5. 인터뷰 질문 (가입 후 1주)

**대면 또는 전화, 15분, 5~10명**

1. 이 서비스를 알게 된 경로는?
2. 가입할 때 hesitated 한 점은?
3. 이음 타입 결과가 본인과 맞다고 느끼나요? (1~5)
4. 추천 모임 중 참여하고 싶은 것이 있었나요?
5. "매칭"이라는 표현 vs "추천" — 어느 쪽이 편한가요?
6. 앞으로 어떤 모임이 있으면 좋겠나요?

---

## 6. 인터뷰 질문 (모임 후 1주)

1. 모임 분위기는 어땠나요?
2. 새로운 분과 대화했나요?
3. 다음에도 참여하실 건가요?
4. 혼자 찾았으면 이 모임을 알 수 있었을까요?
5. 서비스에서 불편했던 점은?

## 8. 정부지원사업 증거 데이터

### 수집 데이터셋

```
이음 타입 → 참여 행동 → 만족도 → 재참여 → (선택) 관계 변화
```

### 리포트에 포함할 지표

| 카테고리 | 지표 |
|----------|------|
| **사회적 고립 완화** | 모임 참여 횟수, 새로운 사람 만남 비율 |
| **디지털 포용** | 50~75세 가입·완료율, 평균 완료 시간 |
| **AI 활용** | Rule-based → 참여 전환율 (Phase 2 AI 비교 baseline) |
| **지역 연결** | 지역별 참여 분포, 모임 개최 수 |

### Project A 연결 (향후)

- 외로움 척도 (UCLA Loneliness Scale) — 사전/사후
- 사회관계 변화 — 인터뷰 + 참여 빈도
- AI 추천 개선 — A/B (rule vs AI)

## 10. Go / No-Go 기준 (6주 후)

### Go (Phase 2 진행)

- 이음 타입 완료율 ≥ 70%
- 모임 참여율 ≥ 40%
- 만족도 ≥ 4.0
- 재참여 ≥ 25%
- 인터뷰에서 "다시 쓰고 싶다" ≥ 60%

### Pivot

- 완료율 < 50% → 질문 수 축소 (12→8)
- 참여율 < 20% → 오프라인 모임 직접 운영 + 앱 보조
- 만족도 < 3.5 → 모임 품질·운영 문제

### Stop

- 가입 < 20명 (4주) → 모집 채널 재검토
- 참여 0건 → 지역/모임 공급 문제
