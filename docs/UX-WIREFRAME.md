# 시니어 UX Wireframe

> 텍스트 와이어프레임. Figma 전 단계 · Sprint 1~3 구현 기준.

---

## 공통 UI 규칙

| 항목 | 값 |
|------|-----|
| 본문 글씨 | 18px (모바일 기준) |
| 제목 | 24~32px |
| 버튼 높이 | 최소 52px (주요 CTA 60px) |
| 버튼 너비 | 화면 90% 또는 full-width |
| 터치 간격 | 16px 이상 |
| 색상 | 브랜드 그린 `#2d7a56`, 배경 `#faf9f7` |
| 한 화면 | **하나의 목적만** |

**금지 UI:** 작은 글씨, 복잡한 탭, 스와이프 필수, "매칭률 87%" 같은 표현

- 12문항 전부 같은 레이아웃
