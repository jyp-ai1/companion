# EPIC 15 — Activity Experience (준비)

## 목표

활동이 "리스트"가 아니라 **하나의 작은 이벤트**처럼 느껴지게 한다.

## 범위 (미구현 — Epic 계획)

| # | 기능 | 설명 |
|---|------|------|
| 1 | 활동 상세 페이지 | 준비물, 소요시간, 난이도, 장소 |
| 2 | 참여자 소개 | Anonymous First 유지 |
| 3 | 날씨 / 장소 정보 | 외부 API 또는 정적 힌트 |
| 4 | 참여 전 체크리스트 | Safe Meet Guide 확장 |
| 5 | 활동 종료 후 추억 | 사진 + 후기 (EPIC 14 후기 연동) |
| 6 | 다음 활동 추천 | Together + Discover 연결 |

## EPIC 14와의 관계

- EPIC 14: **신뢰** (후기, Again, 신고/차단)
- EPIC 15: **경험** (활동 자체의 완성도)

EPIC 14 후기 Flow가 EPIC 15 "추억" 기능의 기반이 됩니다.

## 완료 조건 (예정)

- [ ] `/activities/[id]` 상세 페이지
- [ ] Open Activity + Meetup 통합 Activity Experience
- [ ] 60~70대 무설명 사용 테스트 통과

## 우선순위

1. 활동 상세 + 체크리스트 (가장 체감 큼)
2. 참여자 Anonymous 소개
3. 종료 후 추억 (후기 확장)
4. 날씨/장소 (nice-to-have)
